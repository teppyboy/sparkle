Struct ElementHandle

pub struct ElementHandle { /* private fields */ }
Expand description

Represents a handle to an in-page DOM element

ElementHandles are created by querying the page or locator. They point to a specific element in the page and can be used to interact with it.

§Example

// Note: In practice, you'd use Locators instead of ElementHandles
// ElementHandles are lower-level and don't auto-retry
